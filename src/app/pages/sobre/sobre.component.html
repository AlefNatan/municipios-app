Sobre o Projeto

MunicipiosAPP

Este é um aplicativo desenvolvido em Angular que permite visualizar e consultar informações sobre os municípios do estado do Rio de Janeiro. A aplicação consome dados de uma API REST e apresenta as informações de forma organizada e intuitiva.

Funcionalidades

  • Listagem completa de municípios do estado do RJ
  • Exibição de código IBGE de cada município
  • Sistema de paginação para facilitar a navegação
  • Interface responsiva e moderna
  • Integração com API REST

Tecnologias Utilizadas

Frontend

  • • Angular 21
  • • TypeScript
  • • Tailwind CSS
  • • RxJS

Backend

  • • API REST
  • • HTTP Client

Estrutura do Projeto

O projeto segue as melhores práticas do Angular, utilizando:

  • Standalone Components: Componentes independentes e modulares
  • Services: Lógica de negócio e comunicação com API
  • Interfaces: Tipagem forte com TypeScript
  • Signals: Gerenciamento reativo de estado
  • Environment: Configuração centralizada da API
