Sobre o Projeto

MunicipiosAPP

O MunicipiosAPP é uma aplicação desenvolvida como parte de um desafio técnico Fullstack. O objetivo é consumir dados de uma API criada em .NET 10, responsável por listar municípios de qualquer estado (UF) do Brasil, e exibir essas informações em uma SPA construída em Angular 21, de forma clara, organizada e paginada.

Funcionalidades

  • Seleção de estado (UF) para consulta.
  • Listagem dos municípios retornados pela API.
  • Exibição do nome do município e seu código IBGE.
  • Paginação implementada diretamente na API (.NET 10).
  • Controle de página e quantidade de itens por página no frontend.
  • Interface responsiva e simples, ideal para apresentação do desafio.

Tecnologias Utilizadas

Frontend

  • • Angular 21 (SPA)
  • • TypeScript
  • • Tailwind CSS
  • • Signals
  • • Standalone Components
  • • Deploy na Vercel

Backend

  • • .NET 10 (ASP.NET Core Web API)
  • • Endpoint que lista municípios por UF
  • • Paginação com page e pageSize
  • • Retorno padronizado (name e ibgeCode)
  • • Hospedada via Docker no Back4app (b4a.run)

Estrutura do Projeto

A solução é composta por dois projetos independentes:

  • MunicipiosAPI: API REST em .NET 10 que fornece os dados de municípios.
  • MunicipiosAPP: SPA em Angular que consome a API e apresenta as informações ao usuário.

No frontend, foram utilizadas boas práticas modernas do Angular:

  • Standalone Components: estrutura modular e simplificada.
  • Services: camada dedicada para requisições HTTP e integração com a API.
  • Interfaces: tipagem forte para representar os modelos retornados pelo backend.
  • Signals: gerenciamento reativo de estado nativo do Angular, sem a necessidade de RxJS.
  • Environment:configuração da URL base da API diferenciada para desenvolvimento e produção.
